# IS27-ML
# Лабораторная 1: Парсер

**Задача**: Написать программу для краулинга и парсинга данных с выбранного сайта.

## Шаги:
1. Собрать датасет из как минимум 1000 строк с 6 переменными (2 категориальными, 2 количественными целочисленными, 2 дробными).
2. Таргет может быть как категориальным, так и количественным.
3. Использовать библиотеки Python: `requests`, `lxml`, `beautifulsoup`, `multiprocessing`.
4. Рекомендуется хранить изображения и другие файлы отдельно, в формате `csv`.

## Рекомендации:
- Размер датасета от 3000 до 1’000’000 строк.
- Для параллелизации скачивания использовать прокси.


# Лабораторная 2: Обучение с учителем

**Задача**: Исследовать и подготовить датасет из Лабораторной 1, построить модель с учителем.

## Шаги:
1. Построить графики, обработать данные (отсутствующие значения, выбросы).
2. Реализовать baseline-решение.
3. Выбрать модель обучения (линейная регрессия, дерево решений, случайный лес, бустинг и другие).
4. Обучить модель и подобрать гиперпараметры.
5. Измерить качество модели и провести её сравнение с baseline.
6. Реализовать ту же модель самостоятельно.

## Рекомендации:
- Разделить данные на тренировочную, валидационную и тестовую выборки.
- Использовать Jupyter Notebook для визуализаций и метрик.


# Лабораторная 3: Обучение без учителя

**Задача**: Применить методы обучения без учителя к данным из Лабораторной 1.

## Шаги:
1. Изучить и подготовить данные (аналогично предыдущей лабораторной).
2. Выбрать модель обучения без учителя (k-means, DBSCAN, агломеративная кластеризация).
3. Обучить модель, подобрать гиперпараметры и измерить качество модели.
4. Реализовать модель самостоятельно и сравнить её с реализацией из библиотеки.

## Рекомендации:
- Визуализировать данные с помощью PCA или других методов снижения размерности.
- Использовать Jupyter для представления графиков и результатов.


# Лабораторная 4: Тексты

**Задача**: Собрать текстовый датасет и реализовать NLP модель.

## Шаги:
1. Собрать текстовые данные (из социальных сетей, отзывов, комментариев).
2. Создать таргет для задачи классификации или регрессии.
3. Реализовать текстовую модель, обучить её на собранном датасете.
4. Измерить качество модели, использовать методы снижения размерности для визуализации.

## Рекомендации:
- Использовать библиотеки PyTorch или TensorFlow для моделирования.
- Размер датасета должен быть больше 1000 текстов для лучшего обучения.


# Лабораторная 5: Изображения

**Задача**: Собрать датасет изображений и обучить CNN модель.

## Шаги:
1. Собрать датасет изображений (использовать API или спарсить данные с сайта).
2. Реализовать и обучить сверточную нейронную сеть (CNN).
3. Использовать эмбеддинги изображений для их визуализации и анализа.
4. Применить transfer learning с предобученной моделью (например, ResNet).

## Рекомендации:
- Использовать библиотеку PyTorch для реализации модели.
- Размер датасета для классификации должен быть более 1000 изображений на класс.
